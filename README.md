### 教程
1. 选择头像形状（微信为方形、qq、抖音等平台为圆形）。
2. 上传头像，尽量为宽高比1:1的头像（不支持动图）。
3. 选择喜欢的效果图（目前效果图可适配大部分头像）。
4. 完成上述步骤后，对效果图大小或位置不满意；可在左侧白框区域拖动、缩放效果图。
5. 点击预览可展示方形、圆形的成品头像（支持预览大图）。
6. 点击保存图片。
