### 教程
1. 选择头像形状（微信为方形、qq、抖音等平台为圆形）。
2. 上传头像，（不支持动图）。
3. 选择头像框，贴纸可多选。
4. 完成上述步骤后，对效果图大小或位置不满意；可在画布区域微调，调整透明度。
5. 点击保存图片，分享给朋友。
